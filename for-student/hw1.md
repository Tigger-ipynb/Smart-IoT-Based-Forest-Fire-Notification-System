# Algerian Forest Fires

**คำสั่ง:** ให้นิสิตใช้ชุดข้อมูล `Algerian_forest_fires_dataset.csv` ในการตอบคำถามต่อไปนี้ โดยใช้ไลบรารี Pandas สำหรับจัดการข้อมูล และ Seaborn สำหรับการแสดงผลกราฟ

## Data Preparation

1. โหลดไฟล์ `Algerian_forest_fires_dataset.csv` ขึ้นมาใน DataFrame ชื่อ `df` และตรวจสอบจำนวนแถวและคอลัมน์ 

2. ตรวจสอบว่ามีข้อมูลสูญหาย (Missing Value) ในคอลัมน์ใดบ้าง และมีจำนวนเท่าไหร่ในแต่ละคอลัมน์ 

3. สร้างคอลัมน์ใหม่ชื่อ `Date` โดยรวมข้อมูลจากคอลัมน์ `day`, `month`, และ `year` เข้าด้วยกัน จากนั้นปรับประเภทให้เป็น `Timestamp` และตั้งให้เป็น `index` ของ DataFrame

4. ในคอลัมน์ `Classes` ให้ล้างช่องว่าง (whitespace) ส่วนเกินออก และแปลงข้อมูลให้เป็นหมวดหมู่ (Category) โดยมีค่าเป็น 'fire' และ 'not fire'

## Data Analysis

5. แสดงข้อมูลตั้งแต่วันที่ 1 สิงหาคม 2012 ถึง 15 สิงหาคม 2012 

6. ในเดือนที่มีอุณหภูมิเฉลี่ยสูงที่สุด มีจำนวนครั้งที่เกิดไฟป่า (Classes = 'fire') ทั้งหมดกี่ครั้ง 

7. แสดงวันที่มีความเร็วลม (`Ws`) สูงที่สุด 5 อันดับแรก พร้อมระบุค่าอุณหภูมิ (`Temperature`) ในวันนั้น 

8. คำนวณหาค่าเฉลี่ยของดัชนีไฟป่า (`FWI`) แยกตามสถานะ 'fire' และ 'not fire'

9. หาระยะเวลา (จำนวนวัน) ที่ต่อเนื่องกันนานที่สุดที่ไม่มีเหตุการณ์ไฟป่าเกิดขึ้นเลย (Classes = 'not fire')

## Data Visualization

10. แสดงแผนภาพการกระจายตัว (Probability Density Function) ของค่าอุณหภูมิ (`Temperature`) ของข้อมูลทั้งหมด
11. สร้าง Scatter Plot แสดงความสัมพันธ์ระหว่างอุณหภูมิ (`Temperature`) (แกน x) และ ความชื้นสัมพัทธ์ (`RH`) (แกน y) โดยแยกสีตามสถานะ `Classes` 

12. แสดง Boxplot ของค่า `FFMC` (Fine Fuel Moisture Code) แยกตามเดือน (`month`) เพื่อดูการกระจายตัวของค่าความชื้นเชื้อเพลิงในแต่ละเดือน 
13. สร้าง Bar Plot แสดงค่าเฉลี่ยของปริมาณน้ำฝน (`Rain`) ในแต่ละสถานะ (`Classes`) แยกสีตามเดือน 

14. แสดงกราฟความสัมพันธ์ (Relation Plot) แบบ Grid โดยให้แถวแยกตามเดือน และคอลัมน์แยกตามสถานะ `Classes` เพื่อดูความสัมพันธ์ระหว่างความเร็วลม (`Ws`) และดัชนีไฟป่า (`FWI`) 


## ใบ้

### Data Manipulated 
* `pd.read_csv()` / `.head()` / `.info()` / `.shape`
* `.isnull().sum()` / `.dropna()` / `.fillna()`
* `.astype()` / `pd.to_datetime()`
* `.set_index()` / `.reset_index()`
* `.str.strip()` / `.str.contains()`
* `.loc[]` / `.iloc[]` (สำหรับการเลือกช่วงข้อมูลหรือเงื่อนไข)
* `.groupby()` / `.agg()` (เช่น `mean`, `count`, `max`)
* `.sort_values()` (อย่าลืมดู `ascending=False`)
* `.shift()` / `.diff()` (สำหรับคำนวณส่วนต่างระหว่างแถว)
* `.value_counts()`

### Seaborn , Matplotlib (การสร้างกราฟ)
กลุ่มคำสั่งสำหรับการวาดรูปและวิเคราะห์ความสัมพันธ์:
* `sns.displot()` / `sns.kdeplot()` (เน้นดูการกระจายตัว/ความหนาแน่น)
* `sns.histplot()` (กราฟแท่งความถี่)
* `sns.scatterplot()` / `sns.regplot()` (ดูจุดความสัมพันธ์)
* `sns.boxplot()` / `sns.violinplot()` (ดูค่ากลางและ Outliers)
* `sns.barplot()` (เปรียบเทียบค่าเฉลี่ยในแต่ละกลุ่ม)
* `sns.relplot()` (สร้าง Grid หรือแยก Facet ตามหมวดหมู่)
* `hue=` / `col=` / `row=` (Parameters สำคัญสำหรับแยกสีหรือแยกตารางย่อย)

### Logic , Time Series 
* การเปรียบเทียบ: `==`, `>`, `&` (and), `|` (or)
* การคำนวณช่วงเวลา: `.max() - .min()`
* การดึงส่วนประกอบของเวลา: `.dt.month`, `.dt.day_name()`
